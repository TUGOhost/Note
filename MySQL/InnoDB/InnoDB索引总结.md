# InnoDB索引总结
索引是应用程序设计和开发的一个重要方面。若索引太多，应用程序的性能可能会受到影响。而索引太少，对查询性能又会产生影响。
# InnoDB存储引擎索引概述
InnoDB存储引擎支持以下几种常见的索引：
- B+树索引
- 全文索引
- 哈希索引
